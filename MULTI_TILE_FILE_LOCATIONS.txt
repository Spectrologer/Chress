KEY INTERFACES
==============

MultiTileConfig (src/renderers/strategies/MultiTileRenderStrategy.ts):
- spriteKey: string (image key)
- width: number (tiles)
- height: number (tiles)
- tileType: number (TILE_TYPES constant)
- positionFinderMethod: string (MultiTileHandler method name)
- fallbackChar?: string (single character for no image)
- fallbackColor?: string (hex color for fallback)

StructurePosition (src/renderers/MultiTileHandler.ts):
- startX: number (top-left X of structure)
- startY: number (top-left Y of structure)
